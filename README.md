# Playlist-Manager

Aplikacja do zarządzania playlistami muzycznymi (prawdopodobnie z YouTube/Spotify lub lokalnych plików).

## Funkcje
- Dodawanie/usuwanie utworów
- Tworzenie playlist
- Odtwarzanie muzyki
- ... (dodaj co Twoja appka potrafi)

## Technologie
- Frontend: Vite + React + TypeScript
- Backend: Node.js (lub co tam masz w /server)
- Baza danych: Drizzle ORM (widzę drizzle.config.ts)

## Jak uruchomić lokalnie

```bash
# Sklonuj repo
git clone https://github.com/magnus544x/Playlist-Manager.git
cd Playlist-Manager

# Zainstaluj zależności
npm install

# Uruchom w trybie deweloperskim
npm run dev
